# Source documents

Drop your corpus here, then run ingestion (`make`/`docker compose run ingest`
or `python -m rag_agent.ingest`). Supported: `.pdf`, `.md`, `.txt`.

Add your real research:

- `dissertation.pdf`
- `preprint.pdf`

These are **gitignored** (only this README and `sample_synthetic_research.md`
are committed) so private documents never land in a public repository.
